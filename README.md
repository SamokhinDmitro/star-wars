# Проект star-wars
 Данный проект представляет собой простую верстку домашней страницы на чистом html и css  без применения сторонних фреймворков. <br>
Проект включает в себя:
 * gulp task manager
 * sass
 * jQuery

 Реализован следующий JavaScript функционал:
 * слайдер OwlCarousel
